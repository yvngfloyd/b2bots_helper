# B2Bots Helper

Telegram-бот для приёма заявок от потенциальных клиентов.

## Что умеет

- Показывает стартовый экран с кнопками.
- Даёт кнопки на сайт и Telegram-канал.
- Запускает анкету для квалификации лида.
- Собирает ответы пользователя.
- Отправляет владельцу готовую заявку в Telegram.
- Отправляет владельцу уведомление о каждом пользователе, который нажал `/start`.
- Напоминает пользователю завершить заявку картинками: первый раз через час, затем раз в несколько дней.
- Поддерживает обложку через `COVER_FILE_ID`.

## Быстрый запуск

1. Создай `.env` на основе `.env.example`.
2. Установи зависимости:

```bash
pip install -r requirements.txt
```

3. Запусти бота:

```bash
python main.py
```

## Что обязательно заполнить в `.env`

- `BOT_TOKEN` - токен бота от BotFather.
- `OWNER_CHAT_ID` - твой Telegram ID, куда будут приходить заявки.
- `SITE_URL` - ссылка на сайт.
- `TG_CHANNEL_URL` - ссылка на Telegram-канал.
- `BOT_TITLE` - название бота.
- `COVER_FILE_ID` - необязательно. Когда будет обложка, просто вставь file_id.
- `DATABASE_PATH` - путь к SQLite-базе для пользователей и напоминаний. По умолчанию `bot_data.sqlite3`.
- `FIRST_REMINDER_HOURS` - через сколько часов отправлять первое напоминание. По умолчанию `1`.
- `REMINDER_REPEAT_DAYS` - как часто повторять лёгкие напоминания. По умолчанию `3`.
- `REMINDER_CHECK_SECONDS` - как часто бот проверяет, кому пора отправить напоминание. По умолчанию `300`.

## Как узнать свой chat_id

Самый простой способ:

- написать любому боту для определения ID;
- либо временно добавить логирование update и посмотреть свой user id.

## Команды

- `/start` - открыть главное меню.
- `/menu` - открыть главное меню.
- `/cancel` - отменить текущую заявку.

## Деплой

Подходит для Railway, Render, VPS и других обычных Python-окружений.

Для Railway можно запускать командой:

```bash
python main.py
```
